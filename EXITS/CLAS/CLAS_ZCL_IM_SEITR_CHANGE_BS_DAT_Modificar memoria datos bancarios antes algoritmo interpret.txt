class ZCL_IM_SEITR_CHANGE_BS_DAT implementation.
  METHOD if_ex_fieb_change_bs_data~change_data.

* Consultor: David España (Seidor) (04.01.2019)

* Limitamos que únicamente se utilice para el extracto electrónico:

    CHECK c_febko-anwnd EQ '0001'.


* De manera estándar el sistema iguala la fecha de contabilización a la fecha de extracto.
* Se debe considerar la fecha de contabilización igual a la fecha de operación del
* movimiento bancario.

    c_febep-budat = c_febep-bvdat.

* En caso de que un movimiento entrante no tenga el código de operación de parametrizado,
* se pasará en el campo FEBEP-INFO2 (Información Adicional 2) un texto para diferenciarlos.


    IF ( c_febep-vgint EQ 'Z001' OR
         c_febep-vgint EQ 'Z102' ).

      c_febep-info2 = 'Código Operación NO Definido'.

    ELSE.

      c_febep-info2 = 'Código Operación SI Definido'.

    ENDIF.

* Como información adicional se determinará la descripción del Código de Operación
* Común del movimiento en el campo "Información Adicional 1"

    CASE c_febep-vgext(2).

      WHEN '01'.

        c_febep-info1 = 'Talones-Reintegros'.

      WHEN '02'.

        c_febep-info1 = 'Abonares-Entregas-Ingresos'.

      WHEN '03'.

        c_febep-info1 = 'Domiciliados-Recibos-Letras...'.

      WHEN '04'.

        c_febep-info1 = 'Giros-Transf-Traspasos-Cheques'.

      WHEN '05'.

        c_febep-info1 = 'Amortiz.Préstamos-Créditos-Etc'.

      WHEN '06'.

        c_febep-info1 = 'Remesas Efectos'.

      WHEN '07'.

        c_febep-info1 = 'Suscripciones-Div.Pasivos...'.

      WHEN '08'.

        c_febep-info1 = 'Div.Cupones-Prima Junta-Amort.'.

      WHEN '09'.

        c_febep-info1 = 'Operac.Bolsa-Compra/Venta Val.'.

      WHEN '10'.

        c_febep-info1 = 'Cheques Gasolina'.

      WHEN '11'.

        c_febep-info1 = 'Cajero Automático'.

      WHEN '12'.

        c_febep-info1 = 'Tarjetas Crédito-Tarjetas Déb'.


      WHEN '13'.

        c_febep-info1 = 'Operaciones Extranjero'.


      WHEN '14'.

        c_febep-info1 = 'Devoluciones e Impagados'.


      WHEN '15'.

        c_febep-info1 = 'Nóminas-Seguros Sociales'.


      WHEN '16'.

        c_febep-info1 = 'Timbres-Corretaje-Póliza'.


      WHEN '17'.

        c_febep-info1 = 'Intereses-Comisiones-Gastos...'.


      WHEN '98'.

        c_febep-info1 = 'Anulaciones-Correcc.Asiento.'.

      WHEN '99'.

        c_febep-info1 = 'Varios'.


    ENDCASE.

  ENDMETHOD.
